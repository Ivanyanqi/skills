# Install Notes

- This skill targets macOS only.
- If Xcode Command Line Tools are missing, macOS may show a system installer dialog.
- If `iTerm2` has never been launched on the machine, open it once before rerunning the profile setup script.
- The skill intentionally excludes personal runtime blocks such as `conda`, `nvm`, and `bun`.
- The wallpaper file from the source machine is not bundled. The glass-like look comes from blur and transparency settings alone.
